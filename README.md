# AI Briefing System (Will)

## Goal
Daily AI trend tracking with lightweight daily briefs and deeper weekly/technical writeups.

## Structure
- `templates/` reusable markdown templates
- `daily/` daily brief notes
- `weekly/` weekly digest notes
- `deep-dive/` long-form technical/business analyses

## Naming
- Daily: `daily-YYYY-MM-DD.md`
- Weekly: `weekly-YYYY-Www.md` (ISO week)
- Deep dive: `deep-<topic>-YYYY-MM-DD.md`

## Workflow
1. Generate/collect daily signals
2. Produce short daily brief
3. Expand selected topics into deep-dive markdown
4. Roll up into weekly digest

## Publishing split
- HackMD: drafting + quick edits
- GitHub: canonical archive + version history
